# 图书录入系统

一个简单的图书信息录入与管理 Web 应用，支持图书信息的添加、展示、本地存储和删除操作。界面简洁，适合用于图书管理练习或小型应用场景。

## 功能特点

- 📖 图书信息录入（书名、作者、出版社、出版年份、价格）
- 💾 数据持久化存储（使用浏览器的 `localStorage`）
- 🗑️ 支持删除已添加的图书
- 🎨 简洁美观的 UI 设计
- 📱 响应式布局，适配不同设备

## 技术栈

- HTML5
- CSS3
- JavaScript (ES6)
- LocalStorage

## 使用说明

1. 克隆或下载本仓库到本地。
2. 确保项目目录中包含图片文件 `./images/1 (2).png`（若图片缺失，可替换为其他图片或调整样式）。
3. 使用浏览器打开 `index.html` 文件。
4. 在表单中输入图书信息，点击“添加图书”按钮即可将图书添加到下方表格中。
5. 点击表格中“删除”按钮可移除对应图书。
6. 所有数据会自动保存到浏览器的 `localStorage` 中，刷新页面后数据不会丢失。

## 注意事项

- 项目使用了 `localStorage` 进行数据存储，清除浏览器缓存或数据会导致已录入信息丢失。
- 表单中的“出版年份”和“价格”字段为数字输入框，输入非数字内容时会自动校验。
- 每个输入框均设置了 `required` 属性，提交时会进行非空校验。
- 输入框的浮动标签效果依赖 `placeholder=" "` 和 `:placeholder-shown` 伪类，请勿移除。
